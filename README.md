# FocusFlow - Pomodoro & Productivity Web App 🚀

**FocusFlow** es una aplicación web diseñada para combatir la procrastinación y mejorar la productividad personal mediante la técnica Pomodoro[cite: 28, 36]. Este proyecto fue desarrollado como un Trabajo Colaborativo Contextualizado (TCC) para la asignatura de **Interacción Humano-Computador** en la **Universidad de Cartagena**[cite: 4, 8].

## 📋 Características Principales
* **Temporizador Inteligente:** Sesiones de trabajo de 25 minutos con transiciones automáticas a descansos[cite: 47].
* **Gestión de Tareas:** Checklist dinámico para añadir y organizar actividades pendientes[cite: 40, 49].
* **Interfaz Minimalista:** Diseño basado en el estilo **Glassmorfismo** para reducir la carga cognitiva[cite: 39, 57].
* **Persistencia de Datos:** Uso de `LocalStorage` para que tus tareas no se borren al cerrar el navegador[cite: 65, 78].

## 🧠 Principios de UX/UI Aplicados
La aplicación se construyó siguiendo leyes fundamentales de usabilidad para garantizar una experiencia fluida[cite: 36, 66]:
* **Ley de Hick:** Simplificamos las opciones de la interfaz para facilitar la toma de decisiones rápidas[cite: 67, 68].
* **Psicología del Color:** * 🔴 **Modo Trabajo:** Tonos cálidos para estimular el enfoque[cite: 53].
    * 🟢 **Descanso Corto:** Tonos turquesa para relajación inmediata[cite: 53].
    * 🔵 **Descanso Largo:** Azul profundo para tranquilidad extendida[cite: 54].
* **Accesibilidad:** Contraste adecuado y tipografía legible según estándares WCAG[cite: 74, 86].

## 🛠️ Tecnologías Utilizadas
* **HTML5** y **CSS3** (Flexbox/Grid) [cite: 61, 62, 64]
* **JavaScript (Vanilla)** para la lógica del temporizador [cite: 63]
* **GitHub Pages** para el despliegue en línea.
